Project: Object Detection & Tracking (YOLOv8 + Deep SORT)

Root Directory:
.
├── app.py                  # Flask backend with YOLOv8 + Deep SORT + SQLite + screenshot capture API
├── index.html              # Single-page black & navy blue frontend (HTML + CSS + JS)
├── sort.py                 # Tracking / SORT / Deep SORT support logic (if used directly)
├── yolov8n.pt              # YOLOv8 nano model weights (pretrained on COCO)
├── requirements.txt        # Python dependencies for backend
├── screenshots/            # Folder where captured screenshot images are saved (auto-created at runtime)
└── screenshots.db          # SQLite database file storing screenshot metadata (auto-created at runtime)

Notes:
- VIDEO_SOURCE inside app.py can be 0 (webcam) or a path to your MP4 videos (traffic signals, no-helmet, lane violation, theft detection, etc.).
- The frontend calls:
    - /video_feed   → real-time MJPEG stream with detection + tracking
    - /capture      → takes a screenshot of the current frame and stores it in screenshots/ and screenshots.db
    - /health       → simple health-check endpoint for UI status badges
